# vuePress

基于vuePress开发个人文档系统

## 部署到自己的 github pages

- 手动部署（半自动）

 必须在git bash中执行终端执行命令： `yarn d`， 触发 .sh 脚本发布部署。

 ps：
 无论是 windows cmd 还是 powershell，都无法直接运行 bash 脚本文件
 cmd中执行报错：  `Enter: command not found`

- CICD（推荐）

已配置完成CICD自动部署

部署完成后，访问： [线上地址](https://silin001.github.io/vuePress-blog)
